Excercise Requirements

  • Create Account
  • First Name, Last Name, Email Address, Password, Confirm Password
  • Submit button, when clicked executes a validate function. Can be done as an onsubmit in form tag or added as an event listener.
  • Use Bootstrap and jQuery
  • When the user corrects their input mistakes remove any of the error formatting and border color.

Input Validation

  • All fields are required, use JS not required attribute
  • First name fields should not contain a space. Bonus: Use a regex in JS
  • Optional: find a prebuilt regex and use it to validate the email address is proper
  • Password and Confirm password are required and must be ther same. 2 different error messages
  • Password must contain a min of 8 characters and a mas of 20 characters.
  • Password must contain a letter and a number
  • If all validates properly show an alert that says something nice.

Error Messages

  • Each field that has an error us CSS to make the border of the input box red.
  • Under input bos create / insert HTML to say what the error is.
  • Error message should be red and maybe a slightly smaller font
  • Try to make the message properly left aligned with the input box. (optional)
  • Extreme stretch: Execute the password validation check on each key press and alter the error messages as the password is being typed
